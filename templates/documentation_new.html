Introduction

Welcome to the JSON to Video Converter documentation!

This website allows you to convert JSON files to videos. The conversion process takes a JSON file as input and generates a video that visually represents the data in the JSON file.

Usage

To convert a JSON file to a video, follow these steps:

  1. Prepare your JSON file. Make sure it contains the necessary data for visualization. The structure of the JSON file should be compatible with the video generation process.
  2. Open the JSON2Video website in your web browser.
  3. Click on the "Select a file or drop it here" button to select the JSON file you want to convert.
  4. Optionally, customize the video settings such as resolution and extension.
  5. Click on the "Convert" button to start the conversion process.
  6. Wait for the conversion to complete. The generated video will be available for download or upload.
Supported JSON format

The JSON file should adhere to a specific format to ensure successful conversion.

Here is an example:

  {"title": "Test",
  "resolution": "full-hd",
  "width": 1920,
  "height": 1080,
  "fps": 20,
  "elements": [
    {
      "type": "text",
      "text": "Hello world!",
      "start": 0,
      "end": 5,
      "fontSize": 100
    },
    {
      "type": "image",
      "src": "https://innopolis.com/uploads/thumbs/universitet-innopolis-3e9c6a61e7-1a9f4e6652b3a80977c0e9190611e142.jpeg",
      "start": 0,
      "end": 5

    }
  ]
}
Video Settings

The JSON to Video Converter provides options to customize the generated video. You can modify the following settings:

  • Resolution: Choose the desired width and height for the video in pixels.
  • Frame Rate: Specify the number of frames per second for the video.
  • Encoding Format: Select the video encoding format, such as MP4 or WebM, depending on the supported formats by the converter.
  • Quality: Adjust the video quality or compression level.
  • Additional Options: Some converters may offer additional options like adding audio, transitions, or effects to the video.
